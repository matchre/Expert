Clé publique
La clé publique est (n, e).
n est le produit de 2 nombres premiers p et q si possibles grands.
Le but étant que n soit un nombre très difficile à décomposer. e intervient dans le codage.
Il doit être premier avec (p-1)(q-1)=φ(n).

- Déchiffrement RSA
L'objectif est de retrouver la clé secrète à partir de la clé publique.
Il faut donc essayer de "casser" n (décomposer n en un produit de deux nombres premiers p et q).
On obtient alors la clé secrète (n,d) et il est possible de retrouver le nombre original.
Il suffit de réutiliser l'algorithme de chiffrement avec la clé (n,d) : X≡Yd[n].

- Test de Fermat
Ce test utilise le petit théorème de Fermat : Soit a un nombre entier, si p est premier alors ap≡a[p].
On teste donc un nombre n et on souhaite savoir si il est premier.
Ensuite on choisit un nombre a.
D'après la contraposée, si an n'est pas congru à a modulo n alors n n'est pas un nombre premier.
Par contre, la réciproque de ce théorème n'est pas vraie. Toutefois si an≡a[n] on considère que le nombre n est probablement premier.
